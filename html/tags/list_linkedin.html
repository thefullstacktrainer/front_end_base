Apply for Development tier access:
  1. Create an app, add Advertising API, and complete the access form under My Apps > select your app > Products.
  2. LinkedIn will review your application
  3. Check the Developer Portal for the status of your application
  4. If your application is approved, get started. If your Advertising API application is rejected, review common rejection reasons, create a new app, and submit a new application
